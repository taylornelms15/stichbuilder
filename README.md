# StitchBuilder
#### Author: Taylor Nelms (taylornelms15@gmail.com)

Utility to build cross-stich patterns

Developed from `StitchBuilderGraphical.pyproject` in QtCreator

## Installation Instructions

To generate a Windows executable:

### Python >=3.14

In Powershell or similar, execute:
```
package_pyinstaller.bat
```
This uses pyinstaller on the back end

### Python <=3.13

In Powershell or similar, execute:
```
package.bat
```
This uses nuitka on the back end, which is slow to support Python 3.14+
